Snake written in C using ncurses.

## Build and run

### Clone the repository and move into it
```bash
git clone https://github.com/DerIch69420/cnake.git
cd cnake
```

### Build the binary
```bash
make build
```
or
```bash
make optimized
```

### Run
```bash
./bin/cnake
```

## Play

### Controls
- Arrow keys or wasd to change direction
- 'q' to exit
